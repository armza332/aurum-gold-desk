# 🥇 AURUM — ห้องเทรดทอง AI

บอทเทรดทอง **XAU/USD** ส่วนตัว ที่มี "ทีม AI 7 คน" ตรวจกันเองก่อนเข้าทุกไม้
สถาปัตยกรรม 3 ชั้น: **เว็บ (สมอง) → Bridge (สะพาน) → EA (กล้ามเนื้อ)**

> คติ: **"เข้าน้อยแต่เข้าแม่น" ดีกว่า "เข้ารัวแล้วเจ๊ง"**

## ทีม AI ทำงานยังไง
```
SCANNER เจอจังหวะ → HAWK×3 โหวต (ต้อง 2/3) → SAGE ตรวจซ้ำ (VETO ได้)
→ กฎเหล็ก (IRON) หนีบทุกค่า → AURUM ยิงเข้า MT5 → บันทึก+เรียนรู้
```
ไม่ใช่ AI ตัวเดียวมั่นใจเกินไป — เป็นทีมที่แบ่งหน้าที่ + ฝ่ายเสี่ยง veto + กฎเหล็กหนีบ

## 🚀 พร้อมเทรดจริง (ไม่มีเดโม)
เว็บตั้งเป็น **โหมดจริง** (`demoMode:false`) — ไม่มีข้อมูลจำลอง ก่อนต่อ EA จะขึ้น "ยังไม่เชื่อมต่อ"
เปิดใช้ตามขั้นตอนใน **[`SETUP.md`](SETUP.md)** · EA = **v1.2.0** (ตรวจเวอร์ชันได้บนกราฟ + บนเว็บ)

## เริ่มใช้
เปิดเว็บ (โฮสต์แล้ว): <https://armza332.github.io/aurum-gold-desk/>
หรือรันในเครื่อง:
```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1 8745
```
จากนั้นกดปุ่ม **"ยังไม่เชื่อมต่อ"** (มุมขวาบน) วาง /exec URL ของ Bridge → เชื่อมต่อ
(ดูขั้นตอนเต็มใน **[`SETUP.md`](SETUP.md)**) · แตะตัวละครในห้องเพื่อดูว่าแต่ละคนทำอะไร

## ต่อข้อมูลจริง
วิธีง่าย: ใช้ปุ่ม **"เชื่อมต่อ"** บนเว็บ (จำใน localStorage)
หรือแก้ `js/config.js` ตรง ๆ: `bridgeURL: 'https://script.google.com/macros/s/XXXX/exec'`

> โหมดโชว์เคส (ข้อมูลจำลอง) ปิดอยู่ — เปิดได้ด้วย `demoMode: true` ใน `js/config.js`
(ต้องสร้างชั้น 2 + 3 ก่อน — ดู `ARCHITECTURE.md`)
