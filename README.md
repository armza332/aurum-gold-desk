# 🥇 AURUM — ห้องเทรดทอง AI

บอทเทรดทอง **XAU/USD** ส่วนตัว ที่มี "ทีม AI 7 คน" ตรวจกันเองก่อนเข้าทุกไม้
สถาปัตยกรรม 3 ชั้น: **เว็บ (สมอง) → Bridge (สะพาน) → EA (กล้ามเนื้อ)**

> คติ: **"เข้าน้อยแต่เข้าแม่น" ดีกว่า "เข้ารัวแล้วเจ๊ง"**

## ทีม AI ทำงานยังไง
```
SCANNER เจอจังหวะ → HAWK×3 โหวต (ต้อง 2/3) → SAGE ตรวจซ้ำ (VETO ได้)
→ กฎเหล็ก (IRON) หนีบทุกค่า → AURUM ยิงเข้า MT5 → บันทึก+เรียนรู้
```
ไม่ใช่ AI ตัวเดียวมั่นใจเกินไป — เป็นทีมที่แบ่งหน้าที่ + ฝ่ายเสี่ยง veto + กฎเหล็กหนีบ

## เริ่มใช้ (ชั้น 1 — เว็บ)
```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1 8745
```
เปิด <http://localhost:8745> → กดปุ่ม **"▶ จำลองสัญญาณเข้า"** เพื่อดูทีมทำงานเป็นทอด ๆ
แตะตัวละครในห้องเพื่อดูว่าแต่ละคนกำลังทำอะไร

## ต่อข้อมูลจริง
แก้ `js/config.js`:
```js
dataMode: 'live',
bridgeURL: 'https://script.google.com/macros/s/XXXX/exec',
```
(ต้องสร้างชั้น 2 + 3 ก่อน — ดู `ARCHITECTURE.md`)

- ⬜ ชั้น 2 สะพาน (Apps Script) — เฟสถัดไป
